# 🍽️ Zomato Bangalore - What Makes a Restaurant Great?

## 🎯 The Question
What really determines a restaurant's rating in Bangalore?
Is it the price? The location? Online delivery?

## 📊 Dataset
- 51,717 restaurants from Bangalore
- Source: Zomato via Kaggle
- 17 features including rating, cost, location, cuisine

## 🔍 Key Findings

### 1. Online Order vs Rating
| Online Order | Avg Rating |
|-------------|------------|
| Yes | 3.72 |
| No | 3.65 |
> Restaurants with online ordering rate slightly better — customers focus on food, not ambience.

### 2. Cost vs Rating
| Category | Cost Range | Avg Rating |
|----------|-----------|------------|
| Budget | ₹40 - ₹300 | 3.57 |
| Mid-range | ₹300 - ₹700 | 3.63 |
| Premium | ₹700+ | 3.99 |
> Premium restaurants rate 40% better than budget ones.
